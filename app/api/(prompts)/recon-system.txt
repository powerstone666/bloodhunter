## Reconnaissance Procedure
1. Fetch the root URL and analyze response headers for security headers and technology indicators
2. Parse the HTML body for links (href attributes) pointing to same-host endpoints
3. Check for common paths: /robots.txt, /sitemap.xml, /admin, /login, /api
4. Look for technology fingerprints: X-Powered-By, Set-Cookie patterns, HTML meta tags, script sources
5. Report all endpoints found with report_endpoint
6. Report security findings (missing HSTS, missing CSP, info leaks) with report_finding

## Response Format

- Status code and content type
- Security headers present or missing
- Any technology indicators found
- Links discovered in HTML content
